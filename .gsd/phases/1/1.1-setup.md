# Plan 1.1: Project Foundation & Dependencies

## Objective
Initialize the React application structure manually (to avoid interaction issues in existing directory), install core 3D and animation libraries, and configure the global design system (fonts, colors).

## Context
- .gsd/SPEC.md (Tech Stack: React, Vite, GSAP, R3F)
- .gsd/ROADMAP.md (Phase 1)

## Tasks

<task type="auto">
  <name>Initialize Vite React Project</name>
  <files>package.json, vite.config.js, index.html, src/main.jsx, src/App.jsx</files>
  <action>
    Create the standard Vite + React boilerplate manually since the directory is not empty.
    1. Create `package.json` with `react`, `react-dom`, `vite` scripts.
    2. Create `vite.config.js` with `@vitejs/plugin-react`.
    3. Create `index.html` entry point.
    4. Create `src/main.jsx` and `src/App.jsx`.
    5. Run `npm install` to bootstrap.
  </action>
  <verify>npm run dev -- --port 3000 (verify server starts)</verify>
  <done>Server starts, "Hello World" or App component renders.</done>
</task>

<task type="auto">
  <name>Install 3D & Animation Dependencies</name>
  <files>package.json</files>
  <action>
    Install the specified stack:
    - `three`
    - `@react-three/fiber`
    - `@react-three/drei`
    - `gsap`
  </action>
  <verify>npm list three @react-three/fiber gsap</verify>
  <done>Dependencies present in package.json.</done>
</task>

<task type="auto">
  <name>Configure Design System</name>
  <files>src/index.css, src/App.css</files>
  <action>
    1. Import Google Font "Share Tech Mono" or "VT323" (Sci-Fi/Terminal style).
    2. Define CSS Variables in `src/index.css`:
       - `--bg-color`: #0a0a0a (Deep mixed black)
       - `--accent-cyan`: #a5f3fc (Pale Light Blue)
       - `--accent-pink`: #fbcfe8 (Pink)
       - `--accent-magenta`: #d946ef
    3. Reset body margins and set background color.
  </action>
  <verify>Inspect Element on running app to see variables and background.</verify>
  <done>Styles applied, font loaded.</done>
</task>

## Success Criteria
- [ ] Vite server runs without errors.
- [ ] All 3D/Animation libraries installed.
- [ ] Global CSS variables and fonts are active.
