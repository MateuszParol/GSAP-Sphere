# Plan 1.2: Core 3D Environment

## Objective
Implement the immersive 3D scene with the React Three Fiber Canvas, an animated starfield background, and a placeholder central sphere to verify the 3D pipeline.

## Context
- .gsd/SPEC.md (Goal 1: Immersive 3D Visuals)
- src/App.jsx (Entry point)

## Tasks

<task type="auto">
  <name>Create Scene Component</name>
  <files>src/components/Scene.jsx, src/App.jsx</files>
  <action>
    1. Create `src/components/Scene.jsx`.
    2. Use `<Canvas>` from `@react-three/fiber`.
    3. Set camera position to [0, 0, 5] and fov: 75.
    4. Render `<Scene />` inside `App.jsx`, ensuring it takes 100vw/100vh.
  </action>
  <verify>Browser Check: Blank canvas (black background) without errors in console.</verify>
  <done>Canvas renders full screen.</done>
</task>

<task type="auto">
  <name>Implement Starfield Background</name>
  <files>src/components/Environment/Stars.jsx, src/components/Scene.jsx</files>
  <action>
    1. Use `<Stars>` from `@react-three/drei` for quick implementation OR create a custom points system if needed later. Start with Drei's Stars for "Foundation".
    2. Add `<Stars radius={100} depth={50} count={5000} factor={4} saturation={0} fade speed={1} />` to the Scene.
    3. Ensure they rotate slowly (use `useFrame` if manual rotation needed, or `speed` prop).
  </action>
  <verify>Browser Check: Moving stars visible on black background.</verify>
  <done>Stars visible and animated.</done>
</task>

<task type="auto">
  <name>Add Placeholder Sphere</name>
  <files>src/components/Sphere/MainSphere.jsx, src/components/Scene.jsx</files>
  <action>
    1. Create `src/components/Sphere/MainSphere.jsx`.
    2. Return a `<mesh position={[0,0,0]}>` with `<sphereGeometry args={[1.5, 32, 32]} />`.
    3. Use `<meshStandardMaterial wireframe color="cyan" />` for initial visual check (Cyber aesthetic placeholder).
    4. Add lighting: `<ambientLight intensity={0.5} />` and `<pointLight position={[10, 10, 10]} />` in Scene.
  </action>
  <verify>Browser Check: Wireframe Cyan Sphere visible in center.</verify>
  <done>Sphere renders in 3D space.</done>
</task>

## Success Criteria
- [ ] Full-screen 3D Canvas matches CSS background color.
- [ ] Stars are twinkling/moving in background.
- [ ] Central Sphere is visible is 3D space.
