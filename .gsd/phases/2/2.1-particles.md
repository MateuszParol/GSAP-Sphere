# Plan 2.1: Particle Sphere System

## Objective
Replace the placeholder sphere with a custom particle system. The sphere should be composed of thousands of small square particles (Points) that form a sphere shape. This provides the "low poly / cyber" aesthetic.

## Context
- .gsd/SPEC.md (Goal 1: Immersive 3D Visuals)
- src/components/Sphere/MainSphere.jsx
- Tech Stack: React Three Fiber, Three.js Points
